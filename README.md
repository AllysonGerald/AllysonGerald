# 👨‍💻 Sobre mim

Sou Desenvolvedor Full Stack com experiência na construção de aplicações web, APIs REST e integrações entre sistemas corporativos.

Atuo principalmente com **PHP, Laravel, Symfony, Node.js, NestJS, React e TypeScript**, aplicando princípios como **SOLID, Clean Code, Design Patterns e Arquitetura em Camadas**, sempre buscando código limpo, desempenho e facilidade de manutenção.

Tenho experiência no desenvolvimento de soluções para ambientes corporativos, integração entre ERPs, CRMs e serviços externos, priorizando segurança, escalabilidade e qualidade.
